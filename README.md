Este repositório contém uma série de exercícios práticos desenvolvidos para consolidar os fundamentos da biblioteca **Pandas**, progredindo do básico ao entendimento de estruturas multidimensionais.

## 📖 Conteúdo dos Exercícios

O projeto está estruturado em **5 níveis** de complexidade:

### 1. Nível 1: Estruturas Iniciais
* Criação de uma `Series` vazia.
* Identificação do tipo de dado padrão (`dtype`) gerado pelo Pandas.

### 2. Nível 2: Listas e Indexação Automática
* Conversão de listas Python em `Series`.
* Compreensão do comportamento do índice automático (0 a 4).

### 3. Nível 3: Índices Personalizados (NumPy)
* Integração com a biblioteca **NumPy** para criação de arrays.
* Definição explícita de rótulos (labels) para os eixos (Índices 'A', 'B', 'C', 'D').

### 4. Nível 4: Dicionários e Tratamento de Dados Ausentes
* Uso de dicionários para alimentar `Series`.
* Simulação de valores **NaN** (Not a Number) ao alinhar índices inexistentes.

### 5. Nível 5: Anatomia do DataFrame
* Estudo do construtor `pd.DataFrame()`.
* Definição dos parâmetros essenciais: `data`, `index`, `columns`, `dtype` e `copy`.
